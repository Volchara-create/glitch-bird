# GLITCH BIRD

Психоделічний Flappy Bird на повний екран з глітч-ефектами, які виглядають як реальні проблеми пристрою!

## Що це?

Flappy Bird, але з кожним очком гра стає все більш "зламаною". Гравці реально думають що їхній пристрій зламався!

## Ефекти

- **Повний екран** — гра займає весь дисплей для максимального занурення
- **Ротація як в Geometry Dash** — кожні 10 очок екран повертається на випадковий великий кут (90°, 180°, 270°) — можна грати догори ногами або боком!
- **Chaos Mode (80+ очків)** — ротація змінюється щосекунди, повне божевілля
- **Яскравий психоделічний TV-фон** — насичені фіолетові, зелені, червоні, сині смуги з hue-rotate
- **Темні напівпрозорі труби** — не перетягують увагу від глітч-ефектів фону
- **Скан-лінії + CRT** — ефект старого монітора
- **Реалістичні "системні помилки":**
  - Фейкові Windows-діалоги "System Error"
  - Миттєвий Blue Screen of Death (BSOD)
  - "GPU DRIVER CRASHED", "OUT OF MEMORY"
- **Глітч-ефекти кожні 1-2 сек:**
  - Тремтіння екрану
  - Інверсія кольорів
  - Статичний шум
  - RGB зсув каналів
  - Screen tearing
  - Фріз кадру
  - Горизонтальні глітч-смуги

## Як грати

- **Пробіл** або **Тап** — стрибок
- **Enter** або **Клік** — перезапуск після програшу
- Пташка по центру екрана — уникай труб!
- Чим більше очок — тим більше божевілля!
- Курсор ховається під час гри для повного занурення

## Запуск

```bash
cd flappy
npm install
npm run dev -- --host :: --port 5173
```

## Технології

- React 19 + TypeScript
- Vite 7
- CSS анімації + фільтри (без зовнішніх бібліотек)
- Fullscreen API
- scale(1.42) для покриття екрану при ротації
