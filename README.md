# GLITCH BIRD

Простий Flappy Bird... чи ні? Спробуй набрати якомога більше очок і дізнайся сам.

## Як грати

- **Пробіл** або **Тап** — стрибок
- **Enter** або **Клік** — перезапуск
- Уникай труб і набирай очки
- Що буде далі — побачиш сам...

## Запуск

```bash
npm install
npm run dev
```

## Технології

- React + TypeScript
- Vite
- CSS анімації
